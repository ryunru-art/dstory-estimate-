📋 구글 시트 연동 방법

STEP 1 — 구글 시트 Apps Script 설정

연동할 구글 시트를 열고 상단 메뉴에서 확장 프로그램 → Apps Script를 클릭하세요.

STEP 2 — 아래 코드 붙여넣기

Apps Script 편집기에 기존 내용을 지우고 아래 코드를 붙여넣으세요.

function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getDataRange().getValues();
  var headers = data[0];
  var rows = data.slice(1).map(function(row) {
    var obj = {};
    headers.forEach(function(h, i) { obj[h] = row[i]; });
    return obj;
  });
  return ContentService
    .createTextOutput(JSON.stringify({data: rows}))
    .setMimeType(ContentService.MimeType.JSON);
}
STEP 3 — 웹앱으로 배포

상단 배포 → 새 배포 클릭 → 유형: 웹 앱 → 액세스: 모든 사용자 → 배포 클릭

생성된 웹 앱 URL을 복사하세요.

팀원별 구글 시트 URL을 입력 후 저장하세요. 이후 동기화 버튼으로 데이터를 불러올 수 있습니다.
